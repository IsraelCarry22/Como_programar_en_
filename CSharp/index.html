Como programar en C#


¿Qué es C#?
C# se pronuncia como "C-Sharp". Es un lenguaje de programación orientado a objetos proporcionado por Microsoft que se
ejecuta en .NET Framework y .NET Core.
Con la ayuda del lenguaje de programación C#, podemos desarrollar diferentes tipos de aplicaciones seguras y robustas:

• Aplicaciones de ventana.
• Aplicaciones web.
• Aplicaciones distribuidas.
• Aplicaciones de servicios web.
• Aplicaciones de bases de datos, etc.

C# está aprobado como estándar por ECMA e ISO. C# está diseñado para CLI (Infraestructura de Lenguaje Común). CLI es
una especificación que describe el código ejecutable y el entorno de tiempo de ejecución.
El lenguaje de programación C# está influenciado por los lenguajes C++, Java, Eiffel, Modula-3, Pascal, etc.

En esta extensa documentación se incluye todos los temas de C#, como el primer ejemplo, declaraciones de control, objetos
y clases, herencia, constructor, destructor, esto, estático, sellado, polimorfismo, abstracción, clase abstracta, interfaz, etc,
espacio de nombres, encapsulación, propiedades, indexador, matrices, cadenas, regex, manejo de excepciones,
multithreading, Archivo IO, Colecciones, etc.
